Data Types Foundation

Section Goal

  • Identify the three main data types used in psychology research
  • Recognise which statistical operations each type supports

Before diving into SEM, you need a solid understanding of data types. The type of data you collect determines which statistical tests you can use.

Continuous Data

Numbers with meaningful intervals and distances between values.

  • Age in years (18, 25, 42)
  • Test scores (67, 89, 92)
  • Reaction time in milliseconds (234, 456)
  • Depression score on the BDI (0–63)
  • Height, weight, temperature

Can calculate mean and standard deviation.

Categorical Data

Categories or groups with no inherent numerical meaning.

  • Gender (Male, Female, Non-binary)
  • Diagnosis (Depression, Anxiety, PTSD)
  • Treatment group (Control, Experimental)
  • Ethnicity, occupation, eye colour
  • Likert scales (Strongly Disagree … Strongly Agree)

Cannot calculate a meaningful mean.

Dichotomous Data

A special case of categorical data with exactly two categories.

  • Pass / Fail
  • Yes / No
  • Improved / Not Improved
  • Present / Absent
  • Alive / Deceased

Can be coded as 0/1 for some analyses.

Quick Check

Key Takeaway

The type of data you have directly determines which statistical tests are appropriate. Always identify your data types before choosing an analysis method.

Understanding Variables

Section Goal

  • Distinguish between independent and dependent variables
  • Understand how mediators and moderators extend basic relationships

Independent Variable (IV)

The variable you manipulate or use to predict an outcome.

  • Type of therapy (CBT vs control)
  • Dosage of medication
  • Gender, age group
  • Stress level

Also called: predictor, explanatory variable, factor, treatment.

Dependent Variable (DV)

The variable you measure or try to predict.

  • Depression score after treatment
  • Test performance
  • Recovery rate
  • Reaction time

Also called: outcome, response variable, criterion.

Remember: IV → DV — “Does the IV affect or predict the DV?”

Example: Does therapy type (IV) affect depression scores (DV)?

Interactive: Variable Relationships

Key Takeaway

Research designs always start with identifying which variables are predictors (IVs) and which are outcomes (DVs). Mediators and moderators add nuance to that basic relationship.

Parametric vs Non-Parametric Tests

Section Goal

  • Know when to use parametric versus non-parametric tests
  • Navigate a decision tree to choose the right statistical test

Parametric Tests

Tests that make assumptions about the underlying population distribution.

Requirements

  • Data is normally distributed
  • Data is continuous (scale/interval/ratio)
  • Equal variances (homogeneity)
  • Independent observations

Common Tests

  • Independent t-test
  • Paired t-test
  • ANOVA
  • Pearson correlation
  • Linear regression

More powerful when assumptions are met.

Non-Parametric Tests

Distribution-free tests that require fewer assumptions about the data.

When to Use

  • Data is NOT normally distributed
  • Data is ordinal (ranked)
  • Small sample sizes
  • Outliers present
  • Parametric assumptions violated

Common Tests

  • Mann-Whitney U
  • Wilcoxon signed-rank
  • Kruskal-Wallis
  • Spearman correlation
  • Chi-square

More robust but less powerful.

Interactive: Which Test Should I Use?

Pro Tip: Always check your assumptions BEFORE choosing your test!

Key Takeaway

Parametric tests are more powerful but require stricter assumptions. When those assumptions are violated, non-parametric alternatives give more trustworthy results.

Introduction to SEM

Section Goal

  • Define Structural Equation Modeling and its purpose
  • Understand where SEM fits in the history of statistical methods

Structural Equation Modelling (SEM) is a statistical approach for exploring and analysing relationships between observed variables and underlying latent constructs. It combines multiple regression, factor analysis, and path analysis into a single framework.

History and Evolution

1920s
Path Analysis

Sewall Wright develops path analysis, laying the groundwork for modelling directional relationships between variables.

Mid-20th C
Factor Analysis Evolution

Researchers refine techniques for identifying latent constructs from observed measurements.

1970s
SEM Formalised

Path analysis and factor analysis are unified into the framework now known as Structural Equation Modeling.

1970
EQS Software

Peter Bentler develops the EQS software package, making SEM accessible to applied researchers.

Karimi, L., & Meyer, D. (2014). Structural Equation Modeling in Psychology: The History, Development and Current Challenges. International Journal of Psychological Studies, 6(4), 123-133.

Key Takeaway

SEM builds on decades of statistical innovation. It unifies path analysis and factor analysis, letting researchers test entire theoretical models in a single framework.

Key Concepts and Latent Variables

Section Goal

  • Learn the core terminology of SEM
  • Understand what latent variables are and why they matter

Core Terminology

Latent Variables
Variables not directly observed but inferred from other variables (e.g., intelligence, self-esteem).
Indicators (Observed Variables)
Directly measured variables such as questionnaire items or test scores.
Endogenous Variable (DV)
A variable that is predicted or explained by other variables in the model. Must be continuous.
Exogenous Variable (IV)
A predictor or covariate that is not explained by other variables in the model.
Errors
Variations in observations not explained by the model.
Paths
Hypothesised relationships between variables, shown as arrows in diagrams.
Fit
How well the model represents the observed data.

What Is a Latent Variable?

A latent variable is something we cannot observe directly. It explains patterns in observable (measured) variables.

Example: We cannot measure “burnout” directly, but we can measure exhaustion, detachment, and reduced accomplishment. Burnout is the latent construct; the three dimensions are observed indicators.

Quick Quiz

When and Why Use SEM?

  • Testing complex theoretical models with multiple IVs and DVs
  • Handling latent constructs that cannot be measured directly
  • When multivariate relationships are expected
  • Accounting for measurement error in observed variables
  • Testing multiple relationships simultaneously

Key Takeaway

Latent variables are constructs we infer from measurable indicators. SEM lets us model these hidden constructs alongside directly observed variables in a single analysis.

SEM Analysis Process

Section Goal

  • Know the six steps of a SEM analysis
  • Identify common assumption violations

Steps in SEM Analysis

  1. Model Specification

    Define the theoretical model: which variables, which paths, which latent constructs.

  2. Data Collection

    Gather data with adequate sample size and appropriate measures.

  3. Model Estimation

    Use Maximum Likelihood (ML) or other robust methods (e.g., GLS) to estimate parameters.

  4. Model Evaluation

    Assess fit using multiple indices (Chi-square, CFI, RMSEA, SRMR).

  5. Model Modification

    If fit is poor, revise the model based on theory and modification indices.

  6. Final Interpretation

    Report path coefficients, fit indices, and substantive conclusions.

Assumptions of SEM

  • Adequate sample size — typically 5–10 cases per estimated parameter
  • Multivariate normality — variables should follow a multivariate normal distribution
  • Linear relationships — paths model linear effects
  • No multicollinearity — predictors should not be too highly correlated
  • Correct model specification — the theoretical model should be justified

Activity: Spot the Assumption Violation

Key Takeaway

SEM follows a structured six-step process. Checking assumptions before estimation is critical — violations can lead to misleading results.

Model Fit and Indices

Section Goal

  • Understand what each major fit index measures
  • Know the thresholds for acceptable and good fit

Common Fit Indices

Index What It Measures Good Fit
Chi-square Overall discrepancy between observed and model-implied covariances p > 0.05
CFI Improvement over a baseline (null) model, accounting for sample size ≥ 0.95
RMSEA Error of approximation per degree of freedom, penalises complexity ≤ 0.06
SRMR Average standardised residual between observed and predicted correlations ≤ 0.06

Pro Tip: Never rely on just ONE fit index. Report multiple because different indices are sensitive to different problems.

Interactive: Fit Index Explorer

Practice: Compare Model Fit

Key Takeaway

Model fit is assessed using multiple complementary indices. No single index tells the whole story — always report Chi-square, CFI, RMSEA, and SRMR together.

Software and Tools

Section Goal

  • Know the major SEM software options and their strengths

AMOS

User-friendly graphical interface integrated with SPSS. Great for beginners who prefer drag-and-drop model building.

LISREL

One of the oldest and most established SEM packages. Widely cited in the literature.

Mplus

Flexible and powerful. Handles complex models including mixture models and multilevel SEM.

R (lavaan)

Free and open-source. The lavaan package provides a clean syntax for specifying SEM models in R.

Python (semopy)

Open-source Python package using lavaan-style syntax. Integrates with the broader Python data science ecosystem.

Key Takeaway

Choose SEM software based on your familiarity, budget, and the complexity of your model. Free options like R lavaan and Python semopy are excellent for learning.

Glossary

ANOVA
Analysis of Variance; compares means across three or more groups.
CFI
Comparative Fit Index; ranges 0–1, values above 0.95 indicate good fit.
Chi-square
A test statistic measuring discrepancy between observed and expected covariances.
Continuous Data
Numerical data with meaningful intervals (e.g., age, test scores).
Dependent Variable (DV)
The outcome variable being predicted or explained.
Dichotomous Data
Data with exactly two categories (e.g., yes/no).
Endogenous Variable
A variable predicted by other variables in the model.
Exogenous Variable
A predictor variable not explained by other model variables.
Factor Loading
The strength of association between a latent variable and its observed indicator.
Fit Indices
Statistics that quantify how well a model reproduces observed data patterns.
Independent Variable (IV)
The predictor variable manipulated or measured to explain the DV.
Indicator
An observed variable used to measure a latent construct.
Latent Variable
An unobserved construct inferred from measured indicators (e.g., self-esteem).
Maximum Likelihood (ML)
The standard estimation method in SEM; finds parameters that maximise the likelihood of the observed data.
Mediator
A variable that explains HOW an IV affects a DV (IV → Mediator → DV).
Moderator
A variable that changes the STRENGTH or DIRECTION of the IV–DV relationship.
Multicollinearity
When two or more predictors are very highly correlated, making estimates unstable.
Path Coefficient
The estimated strength of a directional relationship between two variables.
RMSEA
Root Mean Square Error of Approximation; values below 0.06 indicate good fit.
SEM
Structural Equation Modeling; a framework combining factor analysis and path analysis.
SRMR
Standardised Root Mean Square Residual; values below 0.06 indicate good fit.
